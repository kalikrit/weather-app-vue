# Погодное приложение на Vue 3 + TypeScript

Тестовое задание для позиции Frontend-разработчика.

## Описание

Приложение для отображения текущей погоды с автоматическим определением местоположения и возможностью поиска по городам.

**Деплой:** (ссылка, если будете выкладывать)

## Технологии

- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript** — полная типизация
- **Webpack 5** — сборка проекта
- **Axios** — HTTP-запросы к API
- **SCSS** — стилизация
- **ESLint + Prettier** — качество кода

### Дополнительно
- Автоматическое версионирование (скрипт `scripts/update-version.js`)
- Хуки `predev` и `prebuild` для актуализации версии

## Функционал

- Автоматическое определение геолокации при загрузке
- Отображение текущей погоды:
  - Температура
  - Иконка и описание
  - Влажность
  - Скорость ветра
- Поиск города вручную через поле ввода
- Обработка ошибок:
  - Геолокация (отказ пользователя, недоступность)
  - API (неверный ключ, город не найден, ошибки сервера)
  - Сетевые ошибки
- Индикатор загрузки при запросах
- Адаптивная верстка (мобильные устройства + десктоп)

## Установка и запуск

### Требования
- Node.js (версия 16 или выше)
- npm или yarn

### 1. Клонирование репозитория
```bash
git clone https://github.com/ваш-username/weather-app-vue.git
cd weather-app-vue
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта и добавьте:
```env
VUE_APP_WEATHER_API_KEY=ваш_ключ_от_openweathermap
VUE_APP_API_BASE_URL=https://api.openweathermap.org/data/2.5
```
Получить ключ можно бесплатно на [OpenWeatherMap](https://openweathermap.org/api).

### 4. Запуск в режиме разработки
```bash
npm run serve
```
Приложение будет доступно по адресу `http://localhost:8080`.

### 5. Сборка для продакшена
```bash
npm run build
```
Собранные файлы будут в папке `dist/`.

## Версионирование

Проект использует автоматическое управление версиями:

- **Обновление версии**:
  ```bash
  npm run version:patch   # 0.1.0 -> 0.1.1
  npm run version:minor   # 0.1.0 -> 0.2.0
  npm run version:major   # 0.1.0 -> 1.0.0
  ```
- Версия автоматически синхронизируется в `src/version.ts` через скрипт `scripts/update-version.js`.
- Версия отображается в футере приложения.

## Адаптивность

Приложение корректно отображается на:
- Десктопах (ширина > 1024px)
- Планшетах (768px - 1024px)
- Мобильных устройствах (< 768px)

## Тестирование

Откройте консоль браузера — данные погоды для Москвы будут выведены автоматически (тестовый запрос закомментирован в `main.ts`).
